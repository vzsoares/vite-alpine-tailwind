Vite + Alpine + Tailwind

A lightweight SPA starter: pages are plain HTML files, routed entirely on the client with pinecone-router.

Features

Vite

Lightning-fast dev server and optimized builds with HMR and true ESM support.

Alpine.js

Minimal framework for client-side interactivity — the reactivity behind every page here.

Tailwind CSS

Utility-first styling (+ daisyUI components) applied straight to the page markup.
