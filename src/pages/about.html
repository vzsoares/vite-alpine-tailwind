About this template

A lightweight, fully client-side SPA. Each page is a plain HTML file loaded by pinecone-router — no UI framework runtime beyond Alpine, and no build-time prerendering.

The stack

  • Vite for the dev server and optimized builds
  • Alpine.js for client-side reactivity
  • Tailwind CSS v4 (+ daisyUI) for styling
  • Plain HTML pages, routed with pinecone-router
